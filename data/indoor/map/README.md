# Map Images

This directory contains the map images for the indoor scenario.


## File Structure

The map image files should be organized as:
- `MotionState_map_1_xxx.png`
- `GeoLocation_map_2_xxx.png`
- `ArrivalTimeInterval_map_3_xxx.png`
- ... (and so on)

## Note

These files are required for the full functionality of the GTR-Bench assessment system.

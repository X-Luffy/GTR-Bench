# Map Images

This directory contains the map images for the outdoor scenario.

## File Structure

The map image files should be organized as:
- `MotionState_map_259_xxx.png`
- `GeoLocation_map_260_xxx.png`
- `ArrivalTimeInterval_map_261_xxx.png`
- ... (and so on)

## Note

These files are required for the full functionality of the GTR-Bench assessment system.

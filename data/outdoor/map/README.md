# Map Images

This directory contains the map images for the outdoor scenario.

## Download Instructions

Due to file size limitations, the map images are hosted on HuggingFace.

**Download from HuggingFace**: [GTR-Bench Dataset](https://huggingface.co/datasets/your-username/gtr-bench)

After downloading, extract the files to this directory to use the assessment system.

## File Structure

The map image files should be organized as:
- `MotionState_map_259_xxx.png`
- `GeoLocation_map_260_xxx.png`
- `ArrivalTimeInterval_map_261_xxx.png`
- ... (and so on)

## Note

These files are required for the full functionality of the GTR-Bench assessment system.
